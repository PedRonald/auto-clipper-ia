Não. Você só clica no setup.bat e o sistema instala tudo automaticamente. A interface é acessada pelo navegador, como qualquer site. A documentação explica cada passo com print.
Funciona sem placa NVIDIA? +
Sim. Sem GPU NVIDIA, o sistema usa a CPU para encoding. É mais lento, mas funciona perfeitamente. Basta definir USE_NVENC=false no arquivo de configuração.
Para qual tipo de gameplay funciona? +
Qualquer game com picos de energia de áudio — FPS, battle royale, RPG de ação, esportes. Foi testado com God of War Ragnarok. Quanto mais intenso o audio do game, melhor a detecção.
